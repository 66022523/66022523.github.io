ย้อนรอยเหตุการณ์ก่อนที่ผมจะเริ่มต้นบนเส้นทางนี้

  • โรงเรียนเชียงคำวิทยาคม 2561

    Lection - ช่องแรกที่เริ่มต้นและสิ้นสุดลง

    ในช่วงนี้เรียกได้ว่าเป็นช่วงที่ผมเริ่มศึกษาเรื่องของการเขียนโปรแกรมอย่างจริงจังโดยเริ่มที่การเขียนเว็บไซต์พื้นฐานด้วย HTML, CSS, JavaScript (Vanilla.js) และได้คิดที่จะอยากเป็น YouTuber จึงตั้งช่องของตัวเองขึ้นมาชื่อ Lection ซึ่งทำวิดีโอเกี่ยวกับการแก้ไขปัญหาที่พบบ่อยในอปกรณ์อิเล็กทรอนิคอย่างการแก้ไขปัญหา Windows จอฟ้า (BSOD) เป็นต้น ซึ่งช่องดังกล่าวเคยมีเว็บไซต์ที่เขียนขึ้นเองแต่ปัจจุบันได้เก็บไว้และไม่ได้เผยแพร่แล้ว แต่ปัจจุบันนั้นช่องดังกล่าวไม่มีอยู่แล้วเนื่องจากผมคิดว่าน่าจะไม่มีเวลามากพอที่จะมากำกับและดูแล

  • 2562

    Shioru - บอท Discord ตัวแรก

    Discord ในช่วงนั้นที่เริ่มจะได้รับความนิยมมากขึ้นและด้วยความที่อยากหาโปรเจคเล็กๆ ทำเป็นงานอดิเรก ผมได้ไปเห็นว่า Discord นั้นมีบอทให้สามารถสั่งการเจ๋งๆ ได้มากมาย ผมจึงเริ่มสนใจที่จะทำบอทขึ้นมาโดยหาวิธีทำตามในอินเทอร์เน็ตและได้พบวิธีสารพัดมากมาย ผมจึงได้เลือกไลบรารีย์ตัวหนึ่งที่ชื่อว่า Discord.js ผมจึงได้เริ่มศึกษาเกี่ยวกับไลบรารีย์ตัวนี้ตามใน YouTube และเริ่มทำขึ้นมาสักตัว ช่วงนั้นมันทำให้ผมติดการเขียนโปรแกรมงอมแง๋มที่แทบจะไม่หลับไม่นอนเลยก็ว่าได้และหลังจากที่ผมได้เรียนรู้ ผมจึงเริ่มตั้งชื่อบอทตัวของผมเป็นชื่องูๆ ปลาๆ ที่ชื่อว่า Chiye ซึ่งใช้ Heroku เป็นที่โฮตส์ของบอทก่อนภายหลังจะเปลี่ยนชื่อเป็น Shioru ที่คล้ายกับภาษาญี่ปุ่นและใช้เซิร์ฟเวอร์ของ Google Cloud เป็นพื้นที่สำหรับเธอ
  • 2563

    Maseshi - ชื่อที่คุ้นเคย

    หลังจากที่เบื้องหลังได้เริ่มหลากหลายโปรเจคจึงได้มีโปรไฟล์ใหม่เกิดขึ้นบน GitHub ที่ชื่อว่า Maseshi (มาจากคำว่า Masashi ที่แปลว่า ฉลาด) เขามีโปรเจคมากมายที่กำลังดูแลและรักษาอย่างสม่ำเสมอ ในช่วงนี้เขาได้เริ่มทำโปรเจคมากมายไม่ว่าจะเป็น สกินจากเกม osu!: Sakurada Shioro เครื่องมือเปิดใช้งานระบบปฎิบัติการ Windows: WPManager โมเดล ML สำหรับสร้างรูปภาพในรูปแบบการ์ดูนญี่ปุน: Animistatics เครื่องมือสำหรับสร้างรูปภาพด้วย AI: Aicomet (ไม่เปิดเผยในตอนนี้) ถ้าสงสัยว่าเขาคนนั้นคือใคร? เขาคนนั้นไม่ใช่ใครที่ไหนเขาคือผมนั่นเอง 💪

  • 2566 มหาวิทยาลัยพะเยา

    Chasu - เส้นทางที่มิอาจหวนคืน

    ผมได้เริ่มเข้าศึกษาที่มหาวิทยาลัยพะเยาสาขาวิศวกรรมซอฟต์แวร์ ที่ผมเลือกที่มาเรียนสาขานี้เพราะความชอบของผมและผมได้เริ่มเพิ่มนามแฝงให้ตัวเองนั่นก็คือ Chasu ซึ่งมาจากคำนำชื่อสามตัวและนามสกุลหน้าสองตัว สาเหตุที่ผมเลือกที่จะเพิ่มนามแฝงให้ตัวเองนั่นหมายความว่าผมได้ตัดสินใจเดินบนเส้นทางนี้อย่างเต็มตัวแล้ว และผมก็ได้เริ่มโปรเจคอีกหลายๆ อย่างที่สามารถใช้งานได้จริงบวกกับเป็นการดึงศักยภาพในตัวของผมออกมาให้ได้มากที่สุด
